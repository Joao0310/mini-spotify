# 🎵 Mini Spotify API

Projeto desenvolvido em Spring Boot que simula funcionalidades básicas de um sistema de streaming de música, como cadastro de usuários, artistas, álbuns, músicas e playlists.

---

## 🚀 Funcionalidades

- CRUD completo de:
  - Usuários
  - Artistas
  - Álbuns
  - Músicas
  - Playlists

- Regras de negócio:
  - Reprodução de música (incrementa número de execuções)
  - Adição de música em playlist (somente dono)
  - Evita músicas duplicadas na playlist

- Exclusão lógica (soft delete)

## 🛠️ Tecnologias utilizadas

- Java
- Spring Boot
- Maven
